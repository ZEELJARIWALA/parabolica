## 🏁 Experience the Velocity
Parabolica is a premium digital experience designed for the next generation of sim-racing enthusiasts. Featuring a multi-stage, scroll-driven narrative that takes you through the world's most iconic F1 teams.

- **Cinematic Hero Slider**: A tracking-shot experience featuring McLaren, Ferrari, Mercedes, and Red Bull.
- **Scroll-Sync Layout**: All branding and text elements are perfectly synchronized with the user's scroll.
- **VR Integration**: Built-in support for showcasing FPV and Virtual Reality racing setups.
- **Minimalist Design**: A transparent, distraction-free interface focused on the machines and the speed.

## 🛠️ Tech Stack
- **Next.js 15+**: Modern core with high-performance rendering.
- **Framer Motion**: Smooth tracking shots and scroll-linked animations.
- **Tailwind CSS**: Modern, utility-first styling.
- **Lenis**: Silky-smooth professional scrolling.

## 🚀 Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the engine: `npm run dev`
